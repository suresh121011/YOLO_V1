## V1 — Foundation (Months 1–3)

- 23-class YOLO11n model trained on Indian-home data
- Real-time detection at ≥15 FPS on Android
- Voice alerts in Indian English via Piper TTS
- 6 configurable safety rules
- Structured event logging
- 100% offline operation
- Field-tested in 3+ Indian homes

## V2 — Enhancement (Months 4–6)

- Expand to 35+ classes (smoke, fire, stairs, spill, railing, fan, heater)
- Hindi TTS support (Piper Hindi voice model)
- Caregiver mobile dashboard (WiFi sync of event logs)
- Multi-person tracking
- Activity recognition (sitting, standing, cooking, sleeping)
- Enhanced privacy: AES-256 log encryption
- DPDP compliance audit

## V3 — Scale (Months 7–12)

- Smart glasses hardware integration
- Medication schedule management
- Emergency auto-call to caregiver
- Regional language support (Tamil, Telugu, Bengali, Marathi)
- On-device personalization fine-tuning
- Cloud caregiver dashboard
- Fall detection via pose estimation
